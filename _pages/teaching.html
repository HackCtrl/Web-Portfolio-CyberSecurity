Проект

CyberWhale Universe — экосистема практического обучения кибербезопасности

MVP в разработке. Фокус — реальная практика, проверяемые навыки и результат, а не формальные сертификаты.

Автор концепции Архитектор и разработчик MVP Активная разработка

Роль и статус

Роль
Автор концепции, архитектор и разработчик MVP
Статус
Активная разработка
Реализованы foundation, CTF-модуль, админ-панель и publish flow

Видение

Создать end-to-end платформу, где человек проходит путь: обучение → практика на реалистичных стендах → подтверждение навыков → готовность к работе.

Проблема рынка, которую решает проект

Новички в ИБ
Нет понятного маршрута «с нуля до junior»
Разрозненные курсы не дают цельной траектории
Практикующие
Нет безопасных полигонов для практики
Нужны реалистичные стенды без риска для продакшна
Бизнес
Дефицит специалистов с реальными навыками
Сертификаты без практики не закрывают риск

Архитектура платформы (4 модуля)

Learning
Обучающие пути и лаборатории
CTF / Raids
Практические кейсы на Docker-стендах
Skill Passport
Фиксация подтвержденных навыков
AI Assistant
Автоматизация обучения и практики

Что уже реализовано в MVP

  • Архитектурный foundation: frontend shell + backend API + Docker.
  • Админ-панель управления CTF-кейсами.
  • Полный publish flow: Admin → API → Public CTF.
  • Подключение лабораторных стендов через GitHub Releases.
  • Первые Web-кейсы с изолированными стендами.
  • Логика отображения кейсов, подсказок и загрузки стендов.
  • Структура Learning Paths, связанная с CTF.

Ключевая инженерная идея

Не просто «курсы», а связка: теория → практика на уязвимом стенде → закрепление навыка → фиксация в профиле. Каждый кейс — это разворачиваемая лаборатория.

Дорожная карта

  • MVP: Learning + CTF + Admin + стенды.
  • Расширение: Skill Passport + карьерный модуль.
  • B2B-направление: корпоративные полигоны (WhaleGuard).

Ценность проекта

  • Практика вместо теории.
  • Реальные навыки вместо сертификатов.
  • Автоматизация обучения через платформу.
  • Подход «учись, ломай, фиксируй навык».
